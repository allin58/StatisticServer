В данном проекте реализован сбор статистики посещаймости WEB-сайта.
Для простоты развёртования проекта используеться embedded база данных - H2.
Для сборки проекта требуется запустить скрипт build.bat(находится в папке с проектом)
Для запуска - start.bat
Общение с сервером осуществляется посредством GET запросов
http://localhost:8080/event?userId=user1&pageId=page1 (создаёт новое событие)
http://localhost:8080/statistic?startTime=2020-08-02 11:46:48.327&endTime=2020-08-02 13:15:48.327 (запрос статистики за промежуток времени)
Дата вводится в формате 2020-08-02 11:46:48.327 
В случае неправильного ввода даты - сервер ответит статусом bad request 
К базе данных можно получить доступ через веб-интерфейс http://localhost:8080/h2-console (только когда проект запущен)
